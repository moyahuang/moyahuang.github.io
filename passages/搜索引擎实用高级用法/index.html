搜索引擎实用高级用法

发布 : 2019-12-27 浏览 :

Google

运算符 用途 实际用法实例
“”(双引号) 搜索词放在双引号中，代表完全匹配搜索 “keyword”
*（通配符） 星号代表任何文字，比如想查找“搜索*擎”，返回的结果就不仅包含“搜索引擎”，还包含了“搜索收擎”，“搜索巨擎”等内容。 ”搜索*擎“
+（与） 搜索所有包含关键词“keyword1”和“keyword2**”的中文网页 “keyword1 keyword2” 或者“keyword1+keyword2”
-（非） 搜索去除keyword2的keyword1 “keyword1 -keyword2”
OR(或) 搜索 A 或者 B “keyword1 OR keyword2”
~keyword1 搜索它的同义词 ~喜欢
site: 搜索指定域名的内容 keyword site:www.github.com
filetype: 限制搜索的词语是网页中链接内包含的关键词 filetype:pdf ios开发
inanchor:keyword 指令返回的结果是导入链接锚文字中包含搜索词的页面 inanchor:关于我们
-filetype: -filetype:SOMEFILETYPE = 剔除指定文件类型. -filetype:SOMEFILETYPE
allinanchor: 限制搜索的词语是网页中链接内包含的关键词（可使用多个关键词） allinanchor:keyword1 keyword2
allintext: 限制搜索的词语是网页内文包含的关键词（可使用多个关键词） allintext:keyword1 keyword2
intext: 限制搜索的词语是网页内文包含的关键词

测试过~查找同义词，不太好用。符号类就不说了，常用高级查询关键词也就四个：site, filetype, inanchor, intext。

GitHub

GitHub的搜索功能真的超级强大。搜索的范围不仅限于仓库名和README，包含整个仓库！

仓库搜索

语法 含义
stars:>=100
stars:10…50		 限定星数
created: 2017-01-01 限制创建时间在某时间之后
-language: Objective-C 限制开发语言
user: jquery 查找某用户所有仓库
NOT cat 排除所有包含某单词的仓库

文件搜索

语法 含义
cat repo:facebook/react 在仓库facebook/react中搜索包含内容cat的文件
examples path:/docs/ 在指定路径下
join extension:java 在扩展名.java的文件中查找包含内容join的文件
